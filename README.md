## Project architecture

The app has two instances for tickers of the two currencies. In these you can only enter Latin letters. You can change the direction of the search by pressing â‡„ button between two inputs. The currencies data updates every 10 seconds. There are 6 types of historical charts available, varying by time frames.

The application has mobile adaptive layout.

If you have "Data request limit reached for today" error, you can switch apyKey in ./src/assets/data.ts by change "index" variable (only 0-2 are available)
